#  gradient-bg-remover
useful for removing blackground off illustrations

## Requirements
the followings are required for the program to work:
- Python 3
- Pillow

## Usage
```
$ remove_bg.py path_to_image [white_level, black_level]
$ remove_bg.py -h|--help
```
